👋 Welcome to CodeScope

Quick Start Guide:

  1. Navigate to any webpage with code (GitHub, CodePen, etc.)
  2. Select and highlight some code
  3. Right-click → "Analyze Code with AI"
  4. View detailed analysis instantly!

✨ Features: Code summaries, Big O complexity analysis, and visual flowcharts powered by Chrome's on-device AI model

⚠️ First time? Make sure to enable the AI model:
• chrome://settings/ai → Enable "Use AI"
• chrome://components → Update "On-Device Model"
